# DevOps Project 🚀

## 📌 Overview
This is a full-stack project built using Node.js and Express, deployed on Railway.

## ⚙️ Tech Stack
- Node.js
- Express
- HTML, CSS, JavaScript
- Docker
- Railway

## 🚀 Features
- Static Frontend connected to Backend
- REST API integration
- Cloud Deployment
- CI/CD using GitHub Actions
